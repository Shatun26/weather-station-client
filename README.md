## Installation

### Starting project

1. **Clone repository**:

   ```bash
   git clone --single-branch --branch module git@github.com:Shatun26/weather-station-client.git
   ```

2. **Install dependencies**:

   ```bash
   npm i
   ```

3. **Start module**:

   ```bash
   npm run start
   ```
